SECRET

INTRO

The n2o_secret module provides AES/CBC-128 encoder/decoder.

API

pickle(term()) -> binary().

Encodes the term.

depickle(binary()) -> term().

Decodes the term.

sid(term()) -> binary().

Returns HEX of first 10 bytes of HMAC/SHA256 signed seed with a secret.
