UTF8.JS

INTRO

The utf8.js module provides UTF-8 encoder/decoder. The size of utf8.js is 227 bytes.

API

utf8_enc(x)

Perform UTF-8 encoding.

utf8_dec(x)

Perform UTF-8 decoding.
