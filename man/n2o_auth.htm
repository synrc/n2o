AUTH

INTRO

The n2o_auth module provides EMQ auth hook that perform client's autisubscribe to client topic.

CALLBACK

check(#mqtt_client{},binary(),term()) -> ok | ignore.

The default implementation performs client topic autosubscribe at server side.

This module may refer to: n2o, n2o_proto, n2o_mqtt.
